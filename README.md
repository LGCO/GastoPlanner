# 📅 Calendario Financiero

¡Bienvenido a Calendario Financiero! Esta aplicación web te permite gestionar tus ingresos, gastos y movimientos fijos de manera visual y sencilla, usando un calendario interactivo.

## Características principales

- **Visualización mensual:** Consulta tus transacciones por día y obtén un resumen del mes.
- **Gestión de ingresos y gastos:** Añade, edita y elimina movimientos diarios.
- **Movimientos fijos:** Configura gastos o ingresos recurrentes por día de la semana o día del mes.
- **Balance acumulado:** Calcula tu balance y visualiza el mínimo mensual.
- **Respaldo y restauración:** Descarga tus datos en formato JSON y restaura desde un archivo.
- **Modo ejemplo:** Carga datos de ejemplo para probar la aplicación.
- **Diseño moderno:** Interfaz responsiva con Bootstrap y Bootstrap Icons.

## 💻 Uso

- Selecciona el mes y año para visualizar tu calendario financiero.
- Haz clic en un día para ver, agregar o editar transacciones.
- Configura movimientos fijos desde el botón correspondiente.
- Gestiona tus datos desde la sección "Gestor de Datos".
